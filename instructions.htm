H E X A T I C - R O O M: Instructions

<< Back

  1. Load the plan containing the room or rooms of interest as an image.
  2. Select an area of interest by selecting the Room radio button. This allows to draw a blue rectangle on the canvas region.
  3. It is possible to include restraints on the analysis. This impedes positioning persons on indicated regions. Restrains accounts for architectonic elements, furniture or any other element limiting the positioning of persons on the given area. Select the Restraints radio button and draw as many restraints as desired. Restraints are indicated as red rectangles.
  4. Assign an scale to plan. Select the scale radio button and draw a line on the plan with known dimensions given the scale of the plan. On the scale numerical field insert the number of meters corresponding to the width of the line drawn.
  5. Indicate the number of individuals on the numerical field. Note that one could incorporate as many persons as desired, but a 2 meters separation is not granted.
  6. Click on the Analysis button to determine an optimal positioning of persons. The console shows the distance between individuals attained and the maximum number of persons necessary to keep a 2 meters separation between them.
  7. Download the image with the resulting arrangement. Due to the stochastic nature of the procedure it is convenient to run multiple analysis, saving the different configuration and selecting the more convenient for the user purposes
